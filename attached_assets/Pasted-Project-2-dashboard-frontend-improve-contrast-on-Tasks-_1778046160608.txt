Issue:
Tasks and Settings are too low-contrast.
Everything is too pale white/grey, making it hard to distinguish sections, cards, controls, and action areas.

Task:
Improve visual contrast and hierarchy on:
1. Tasks page
2. Settings page

Goal:
Keep the premium SaaS style, but make the pages clearer, easier to scan, and more visually separated.

Do not make it dark mode.
Do not use loud colors everywhere.
Keep it clean, premium, and professional.

==================================================
PART A — TASKS CONTRAST
==================================================

Improve separation between:
- task composer
- sync/status bar
- filter tabs
- loading state
- task cards
- task footer/actions

Required Tasks changes:
1. Increase contrast between page background and cards.
2. Give task cards stronger but still tasteful borders.
3. Make composer visually distinct from task cards.
4. Make the sync/pending status bar clearer.
5. Make filter tabs more readable and visually anchored.
6. Make the loading state less dominant or visually softer.
7. Make task card footer/action area clearly separated.
8. Make “Pending sync” badge visible but elegant.
9. Text contrast should be stronger, especially grey helper text.

Suggested Tasks visual style:
- Page background: slightly darker soft grey, e.g. #f6f8fb or #f7f8fa
- Cards: pure white
- Card border: #d9dee7 or #dfe3ea
- Active task card border: slightly stronger #cfd6e3
- Composer border: #cfd6e3
- Footer/action area: #f8fafc with top border #e2e8f0
- Loading box: lighter, smaller, not huge dashed box if possible
- Text:
  - main text: #1f2937 / #202124
  - secondary: #4b5563 instead of very pale grey
  - helper: #6b7280
- Pending sync badge: soft amber background with darker amber text
- Open/Done/All tabs: stronger active state, clearer inactive text
