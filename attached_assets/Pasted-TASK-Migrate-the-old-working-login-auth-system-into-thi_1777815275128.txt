2. Create AuthProvider.tsx:

- use localStorage token key:
  `wtyj_token_${localStorage.getItem("wtyj_client") ?? "default"}`

- On login:
  - store token in localStorage under current client token key
  - update React state

- On logout:
  - remove current token key
  - remove wtyj_client
  - set token state null
  - navigate to /login

- Register API unauthorized handler:
  setOnUnauthorized(() => {
    clear auth
    navigate("/login", { replace: true })
  })

IMPORTANT:
AuthProvider must be rendered inside BrowserRouter because it uses useNavigate().

3. Create ProtectedRoute.tsx:

4. Update api.ts auth logic:

- VALID_CLIENTS:
  ["bluemarlin", "adamus", "consultadespertares", "unboks"]

- getClient()

    https://api.wetakeyourjob.com/{client}/dashboard/api

- BASE_URL:
  initialized using getClient()

- setOnUnauthorized(fn)

- 401 guard:
  - count 401s within 60 seconds
  - first 401 can be ignored if token exists
  - second 401 clears auth
  - if no token exists, clear auth immediately

- api.login(password)

  - sends { password }
