Issue:
The current Translate control on every message is too large and repetitive.

Current control:
[Translate] [Dutch ▼]

It takes too much space inside the conversation thread.

Also, operators need a faster way to translate the whole conversation instead of clicking Translate on every single message.

Task:
Improve translation UX.

Required features:
1. Make per-message Translate controls smaller.
2. Add a “Translate all” control for the whole conversation.
3. Add a checkbox/toggle option for showing translated messages.
4. Keep selected target language shared globally across all messages.
5. Default target language remains English unless user changed it.
6. If user chooses Dutch, Translate all translates all visible messages to Dutch.
7. Do not send anything to the customer.
8. Do not modify original message history.

Header/control placement:
Add a compact translation toolbar near the conversation header or just above the message list.

Suggested layout:
[ ] Translate all messages   Language: [Dutch ▼]

or:
[Translate all] [Dutch ▼] [✓ Show translations]

Preferred premium layout:
- Small compact toolbar
- Light background
- Subtle border
- Does not take much vertical space
- Sits above messages, below the contact header/mode controls

Required behavior:
1. “Translate all” translates all visible messages in the current conversation.
2. It should use the selected global language.
3. It should skip messages already translated to that language.
4. It should show progress/loading state:
   “Translating…”
5. If some translations fail, keep successful translations and show calm note:
   “Some messages could not be translated.”
6. The operator can hide/show translations.
7. A checkbox/toggle controls whether translations are displayed:
   - checked = show translations
   - unchecked = hide translations
8. Per-message “Hide translation” still works if already present, but global show/hide should be respected.
9. Translation cache must be by message id + target language.
10. Changing language updates all controls and future translations.

Per-message button changes:
- Make button smaller.
- Do not use a large pill for every message.
- Use compact text/icon:
  “Translate”
  or small icon + “Translate”
- Language dropdown should not repeat on every message if global toolbar exists.
- Preferred:
  - Global toolbar has language selector.
  - Per-message button only says “Translate”.
